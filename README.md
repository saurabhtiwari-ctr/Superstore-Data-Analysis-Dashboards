## Questions (KPIs)
-	Total sales in each Ship Mode.
-	Maximum sales in each Category and Sub-Category.
-	Region wise sales in each Category.
-	Relation between Profit, Quantity, Discount and Sales plot.
-	Total Profit in each region.
-	Which State have Maximum Profit.
-	Total profit in each Category and Sub-Category.
-	Total Quantity in each Category and Sub-Category.
-	Relation between Sales and Profit in Region wise.
-	Relation between Discount & Quantity in Region wise.
-	Dashboard Interaction <a href="https://github.com/saurabhtiwari-ctr/Superstore-Data-Analysis-Dashboards/blob/main/Superstore%20Sales%20project%20using%20Python.pdf"  >View Dashboard</a>

## Process
-	Verify data for any duplicate values, improper data, date related issues, drop the column witch     are not useable for analysis and extra space.
-	Make sure data is consistent and clean with respect to data type, format and value used.
-	Create plot with the help of Seaborn, Matplotlib on the bases of question asked.
-	Merge all plot into one dashboard for better understanding.

## Project Insight
-	Standard Class have maximum sales in ship mode (1358215.7).
-	Chairs have maximum sales about (328449.103) in sub-category
-	Technology have maximum sales in each category (836154) and profit ( 145000).
-	West region have maximum profit (110000)
-	Maximum Negative profit(loss) in Tables (-18000) and maximum profit in Copiers (55000).
-	Maximum quantity (6000) are Binders and minimum (200) are Copiers.
-	Office supplies category  have maximum quantity (22500).
-	Maximum sales and profit are in west region.

## Final Conclusion
Standard class of ship mode, Phone of Sub-category, and Technology in category have maximum sales. West region, copiers in sub-category, and technology have maximum profit.
Maximum loss in table of sub-category. Binders in sub-category, and office supplies have maximum quantity. West region have maximum sales and profit.
